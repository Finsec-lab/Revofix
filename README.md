### Install

```
adb install -r Revofix.apk
```

Open the app → **Grant** → **Install** → **Reboot**.

&nbsp;

### Requires

Magisk, KernelSU or APatch · Android 8 +

&nbsp;

### Privacy

No permissions. No network. Per-process scope.
